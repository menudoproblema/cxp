# Catálogo de HTTP Transport

## Interfaz
Este catálogo define el vocabulario canónico de capabilities para la interfaz `transport/http`.

## Capabilities
- `headers`
- `cookies`
- `auth_propagation`
- `middleware_observable`
- `streaming`
- `network_transport`

## Tier Núcleo
El tier `core` exige:

- `headers`
- `network_transport`

## Procedencia
Este catálogo nace del vocabulario ya presente en Cosecha para perfiles de runtime HTTP.
