# Catálogo de HTTP Application

## Interfaz
Este catálogo define el vocabulario canónico de capabilities para la interfaz `application/http`.

## Capabilities
- `routing`
- `schema_validation`
- `dependency_injection`
- `auth_processing`

## Tier Núcleo
El tier `core` exige:

- `routing`
- `schema_validation`

## Procedencia
Este catálogo nace del vocabulario ya presente en Cosecha para aplicaciones HTTP.
