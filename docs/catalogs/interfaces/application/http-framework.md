# Catálogo de HTTP Application Framework

> Estado actual: catálogo de compatibilidad para la semántica anterior de framework HTTP publicada bajo `application/http`.

## Interfaz
Este catálogo define el vocabulario canónico de capabilities para `application/http-framework`.

## Capabilities
- `routing`
- `schema_validation`
- `dependency_injection`
- `auth_processing`

## Tier Núcleo
El tier `core` exige:

- `routing`
- `schema_validation`

## Procedencia
Este catálogo preserva el vocabulario ya presente en Cosecha para aplicaciones HTTP de alto nivel.
